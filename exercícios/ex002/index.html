Paragráfos e Quebras de Linhas

Você pode escrever absolutamente qualquer coisa dentro das tags <p> e </p> (com excessão do nome das próprias tags... por enquanto). Inclusive, não importa a quantidade de Enteres ou espaços entre as palavras. O texto sempre sairá normalmente se você não utilizar tags especificas para alterá-lo.

Se quiser fazer uma quebra de linha dentro de um parágrafo, deve-se usar a tag <br>.


Exemplo:

Olha a quebra de linha vindo...
...veio

No caso de querer usar multiplas quebras de linhas, favor nunca utilizar multiplas tags <br>
